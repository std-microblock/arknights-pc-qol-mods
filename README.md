### 功能说明
#### 帧率解锁
自动解锁帧率限制至 144 帧

#### 划火柴
使用方法同模拟器。

由于 PC 版限制（没有触屏功能），无法做到同一帧内部署太多干员。

本功能可以在同一动画帧内部署 3~5 个干员。
